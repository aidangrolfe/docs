---
title: "Get guest fixture snapshots"
api: "GET /guest/fixtures/snapshot"
description: "Get the latest snapshots of fixtures upcoming in the next week from now"
---

Get the latest snapshots of fixtures upcoming in the next week from now, corresponding to freely available stable prices in limited competitions.

## Query Parameters

<ParamField query="competitionId" type="int32">
  Optional. Filter by a specific competition ID.
</ParamField>

## Response

Returns an array of Fixture objects.

### Fixture Object Properties

<ResponseField name="Ts" type="int64" required>
  Timestamp of the fixture data
</ResponseField>

<ResponseField name="StartTime" type="int64" required>
  Fixture start time timestamp
</ResponseField>

<ResponseField name="Competition" type="string" required>
  Competition name
</ResponseField>

<ResponseField name="CompetitionId" type="int32" required>
  Competition identifier
</ResponseField>

<ResponseField name="FixtureGroupId" type="int32" required>
  Fixture group identifier
</ResponseField>

<ResponseField name="Participant1Id" type="int32" required>
  First participant identifier
</ResponseField>

<ResponseField name="Participant1" type="string" required>
  First participant name
</ResponseField>

<ResponseField name="Participant2Id" type="int32" required>
  Second participant identifier
</ResponseField>

<ResponseField name="Participant2" type="string" required>
  Second participant name
</ResponseField>

<ResponseField name="FixtureId" type="int64" required>
  Unique fixture identifier
</ResponseField>

<ResponseField name="Participant1IsHome" type="boolean" required>
  Indicates if participant 1 is the home team
</ResponseField>
