---
title: "Get fixture updates for a given hour"
api: "GET /fixtures/updates/{epochDay}/{hourOfDay}"
description: "Get all fixture updates for a single fixture on a given day"
---

Get all fixture updates for a single fixture on a given day.

## Path Parameters

<ParamField path="epochDay" type="integer" required>
  The day since the Unix epoch
</ParamField>

<ParamField path="hourOfDay" type="integer" required>
  The hour of the day (0-23)
</ParamField>

## Response

<ResponseField name="data" type="array<Fixture>">
  A list of fixture updates. If no updates are found for the given parameters, an empty list is returned.
</ResponseField>

### Fixture Object

<ResponseField name="Ts" type="int64">
  Timestamp of the fixture data
</ResponseField>

<ResponseField name="StartTime" type="int64">
  Fixture start time timestamp
</ResponseField>

<ResponseField name="Competition" type="string">
  Competition name
</ResponseField>

<ResponseField name="CompetitionId" type="int32">
  Competition identifier
</ResponseField>

<ResponseField name="FixtureGroupId" type="int32">
  Fixture group identifier
</ResponseField>

<ResponseField name="Participant1Id" type="int32">
  First participant identifier
</ResponseField>

<ResponseField name="Participant1" type="string">
  First participant name
</ResponseField>

<ResponseField name="Participant2Id" type="int32">
  Second participant identifier
</ResponseField>

<ResponseField name="Participant2" type="string">
  Second participant name
</ResponseField>

<ResponseField name="FixtureId" type="int64">
  Unique fixture identifier
</ResponseField>

<ResponseField name="Participant1IsHome" type="boolean">
  Indicates if participant 1 is the home team
</ResponseField>
