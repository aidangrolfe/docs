---
title: "Start a new guest session"
api: "POST /guest/start"
description: "Start a new guest session"
---

Initiates a new, anonymous guest session and returns a short-lived JSON Web Token (JWT).
This token must be provided as a Bearer token in the Authorization header for subsequent calls,
such as the token activation endpoint.

## Response

<ResponseField name="token" type="string" required>
  A short-lived JWT for the guest session
</ResponseField>
