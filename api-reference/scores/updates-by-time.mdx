title: "Get score updates from a specific historical 5-minute interval"
api: "GET /scores/updates/{epochDay}/{hourOfDay}/{interval}"
description: "Get a json array of all score updates from a specific historical 5-minute interval"
---

Get a json array of all score updates from a specific historical 5-minute interval.

## Path Parameters

<ParamField path="epochDay" type="integer" required>
  The day since the Unix epoch
</ParamField>

<ParamField path="hourOfDay" type="integer" required>
  The hour of the day (0-23)
</ParamField>

<ParamField path="interval" type="integer" required>
  The 0-indexed 5-minute interval within the hour (0-11)
</ParamField>

## Query Parameters

<ParamField query="fixtureId" type="int32">
  Optional filter by fixture ID
</ParamField>

## Response

<ResponseField name="data" type="array<Scores>">
  Array of Scores objects from the specified time interval
</ResponseField>

### Scores Object

The Scores object contains detailed information about score events and statistics for a fixture. The structure varies by sport type (Basketball, Football, etc.) and includes:

<ResponseField name="FixtureId" type="int64">
  Fixture identifier
</ResponseField>

<ResponseField name="Seq" type="int32">
  Sequence number of the score event
</ResponseField>

<ResponseField name="Ts" type="int64">
  Timestamp of the score event
</ResponseField>

<ResponseField name="Stats" type="object">
  Map of statistical keys to their values
</ResponseField>

<ResponseField name="Sport" type="string">
  Sport type (Basketball, UsFootball, etc.)
</ResponseField>
