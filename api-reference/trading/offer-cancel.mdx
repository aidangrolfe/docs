---
title: "Cancel an open trade offer"
api: "POST /trading/offer/cancel"
description: "Cancel an open trade offer"
---

Allows the original creator of an offer to withdraw it from the system.

The server will verify that the authenticated user is the owner of the specified `offerId`.

This is an atomic operation. The server will only update the offer's status to 'CANCELLED' if the offer's current status is 'OPEN'.

If successful, the endpoint returns a `200 OK` status, and the offer is immediately removed from the pool of available trades.

## Request Body

The ID of the offer to be cancelled.

<ParamField body="offerId" type="int32" required>
  The ID of the offer to cancel
</ParamField>

## Response

<ResponseField name="message" type="string">
  Confirmation that the offer was successfully cancelled
</ResponseField>
